Welcome to the Jenn Gen website!

This is a test website. JennGen is an AI-driven static site generator that transforms your content into high-quality, deployable code.

Getting Started

Prerequisites

  • Node.js - JavaScript runtime
  • npm - Node.js package manager

Running

  1. Get your OpenAI key from https://beta.openai.com/
  2. Use npx to get JennGen and render the current folder
export OPENAI_API_KEY=<YOUR_API_KEY>
npx jenngen .

4. Open http://localhost:3000 to view it in the browser.
